# Continumm

A project with backend, infrastructure, and deployment configurations.

## Structure

- `backend/`: Backend application code
- `infra/`: Infrastructure as code (Terraform, Ansible)
- `deploy/`: Deployment configurations (Docker Compose, Nginx, Prometheus)
- `.github/workflows/`: CI/CD workflows

## Getting Started

1. Build and run the backend:
   ```bash
   cd deploy
   docker-compose up --build
   ```
